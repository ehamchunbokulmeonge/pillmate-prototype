 **PillMate (필메이트)**

📸 사진 한 장으로 약물 정보·중복 복용 위험을 분석하는 AI 헬스케어 서비스



🩺 AI 기반 약물 관리 서비스

사진 촬영 → AI 자동 분석 → 성분·부작용·복용법 제공 → 위험 경고


**🚀 1. 서비스 소개**

PillMate는 약 패키지를 촬영하면 AI가 약 정보를 즉시 분석해주는 서비스입니다.
<img width="585" height="1266" alt="약추가" src="https://github.com/user-attachments/assets/a21383d5-b59d-452f-bf53-d96939ffb59f" />

**💡 2. 왜 필요한가? (배경)**

	•	고령화/만성질환 증가 → 다제약물 복용자 급증
	•	약 종류 많아질수록 → 중복 복용 / 부작용 위험 증가
	•	약사 인력 부족 → 충분한 상담 어려움
	•	기존 앱들은 약 이름 직접 입력해야 함 (불편)

**🌟 5. 기대 효과**

의료·사회적 효과
	•	중복 복용·오남용 사전 예방
	•	약사 상담 부담 감소
	•	고령층·만성질환자의 복약 안전 수준 향상

	•	디지털 헬스케어 산업 경쟁력 강화
