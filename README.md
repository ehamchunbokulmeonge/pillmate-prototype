📌 1. 서비스 개요 (Overview)

PillMate는 스마트폰으로 약 패키지를 촬영하기만 하면
AI가 자동으로 약 이름, 성분, 복용법, 주의사항을 분석하여 제공하는 AI 기반 약물 관리 서비스입니다.

사진 한 장으로:
	•	약 이름·성분 자동 인식(OCR)
	•	위험 성분·중복 복용 감지
	•	복약 스케줄 자동 생성
	•	AI 약사 상담 제공

복약 오류를 줄이고, 안전한 약물 복용 환경을 만드는 것을 목표로 합니다.

📌 2. 제안 배경 (Background)

현대 사회의 고령화·만성질환 증가로 다제약물 복용자가 크게 늘고 있습니다.

문제점:
	•	서로 다른 병원에서 받은 약의 중복 성분 복용 위험
	•	부작용 및 상호작용에 대한 정보 부족
	•	약사 인력 부족으로 충분한 상담 어려움
	•	고령층의 약 정보 검색 난이도 증가

따라서 “사진 한 장으로 약물 안전 관리”가 가능하도록 하기 위해
PillMate를 기획했습니다.


📌 3. 활용 데이터 (Open Dataset)

1. 마약류 약물·오남용 정보
2. 의약품 낱알식별 정보
3. 의약품 제품 허가정보
4. 의약품/화장품 패키징 OCR 데이터
5. 의약품개요정보(e약은요)
6. 경구약제 이미지 데이터


📌 4. 핵심 기능 (Features)

1) 약 패키지 OCR 인식(이미지 → 약 정보 자동 인식)
	•	스마트폰으로 약 포장지 촬영
	•	AI Vision이 약품명/용량/제약사 자동 추출
	•	오타·노이즈 제거 후 정확한 약 이름으로 매핑

2) 복약 안전 분석
	•	성분 중복 여부 감지
	•	병용금기, 주의 성분 자동 분류
	•	부작용·위험 약물 여부 실시간 경고

3) 복용 스케줄 관리
	•	시간 설정 기반 알림
	•	AI 기반 자동 복용 시간표 추천
	•	복용 체크 → 스케줄 자동 조정

4) AI 약사 챗봇
	•	사용자가 등록한 약 목록을 Context로 활용
	•	복용법, 상호작용, 주의사항 답변 제공


📌 5. 적용 기술 (Tech Stack & AI Models)

OCR & Vision
	•	EasyOCR / Tesseract OCR Fine-Tuning
	•	OpenCV 기반 전처리(기울기 보정, 대비/노이즈 개선)

텍스트 정규화
	•	Transformer 기반 약품명 정규화/오타보정 모델

약물 분석 엔진
	•	Rule-Based 성분 매칭
	•	Embedding Similarity 기반 유사 성분 탐지
	•	식약처 DB 기반 약 정보 매핑

AI 상담
	•	GPT 기반 LLM
	•	사용자 약 목록을 컨텍스트로 사용

개발 환경
	•	Backend: FastAPI
	•	Frontend: React / React Native(선택)
	•	DB: PostgreSQL / Supabase
	•	Infra: Docker, AWS or Render


📌 6. 서비스 흐름 (Service Flow)
	1.	사용자가 약 패키지 촬영
	2.	이미지 전처리(OpenCV)
	3.	OCR 통해 텍스트 추출
	4.	Transformer 정규화 → 정확한 약품명 매칭
	5.	식약처 DB로 정보 조회
	6.	성분 기반 위험 분석
	7.	사용자가 약 목록에 저장
	8.	복용 스케줄 설정 및 알림
	9.	AI 약사 상담 가능


📌 7. 서비스의 창의성 및 차별점
	•	기존 약물 앱 = 사용자가 직접 약 이름 입력
	•	PillMate = 사진 한 장으로 모든 약 정보 자동 분석
	•	중복 성분·상호작용 분석 기능은 현재 상용 앱에서도 드물게 제공
	•	고령층도 쉽게 사용 가능
	•	AI Vision + LLM + 의료데이터를 통합한 실제 구현 가능한 구조


📌 8. 기대효과 (Expected Impact)

🌟 사회적 효과
	•	중복 복용·오남용 예방
	•	약사 상담 수요 분산 → 약료 효율성 증가
	•	고령층의 디지털 건강관리 접근성 향상

🌟 경제적 효과
	•	부작용으로 인한 병원 재방문 감소
	•	약물 오남용 관련 의료비·사회비용 절감
	•	디지털 헬스케어 산업 성장 촉진
